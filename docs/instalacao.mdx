# Instalação

Veja essa seção para aprender como instalar o AmazonasDataHub.

## Início

### Requisitos

Para instalar o `amazonasdatahub`, é necessário que você possua
instalado no seu sistema operacional ou ambiente de desenvolvimento:

-   R version 4.41.1 (2024-06-14) – “Race for Your Life”;
-   Pacote `devtools` do R.

## Instalando o devtools

Caso não possua o `devtools` instalado, instale-o usando no terminal do
R:

```r
install.packages("devtools")
```

## Instalando o amazonasdatahub

Ainda no terminal do R, após ter instalado o `devtools`, instale o
pacote `amazonadatahub` através do método `install_github()` do pacote
`devtools`:

```r
library(devtools)
install_github("onelsoncarvalho/amazonasdatahub")
```

## Usando no R

Assim que o `amazonasdatahub` for instalado, use `library()` ou
`require()` para carregar o pacote.

```r
library(amazonasdatahub)
```
