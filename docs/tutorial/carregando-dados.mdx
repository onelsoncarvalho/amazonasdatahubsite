# Carregando Dados

Para usar as bases de dados do AmazonasDataHub, primeiramente, garanta
que o pacote já esteja instalado (veja a aba de Tutorial - Instalação).

Carregue o pacote usando a função `library()`:


```r
library(amazonasdatahub)
```

## Dados do AmazonasDataHub

Até o momento, o pacote `amazonasdatahub` conta com oito bases de dados
para você, estudante, estatístico ou entusiasta, para analisar e aplicar
métodos estatísticos.

- agriculture_amazonas: Plant production data from the Institute of
  Sustainable Agricultural and Forestry Development of Amazonas - IDAM
  (2023);
- aids_amazonas - A dataset of AIDS occurrences from 2011 to 2023 in
  Amazonas;
- gdp_amazonas: Gross Domestic Product - GDP of Amazonas;
- humidity_manaus: Time series with Extreme Values of Limited Range;
- malaria_amazonas: An integrated dataset of malaria notifications in
  the state of Amazonas;
- prf_amazonas: Traffic accident data from Amazonas recorded by Federal
  Highway Police in 2025;
- rionegro_amazonas: A dataset of the Rio Negro River (Amazonas) level
  from 2003 to 2023;
- srl_muni: Physical literacy and reading performance of amazonian
  schoolchildren: an association study.

Assim que o pacote é carregado, essas bases de dados já podem ser
acessadas.

### Documentação

Cada base de dados do `amazonasdatahub` é documentada, explicando
detalhes sobre a base de dados e o que cada variável representa. Para
ver a documentação, basta utilizar o `?` precedido do nome da base de
dados.


```r
?agriculture_amazonas
?aids_amazonas
?gdp_amazonas
?humidity_manaus
?malaria_amazonas
?prf_amazonas
?rionegro_amazonas
?srl_muni
```
